# v0.3.0 Release Notes - World & Exploration

**Release Date:** October 30, 2025  
**Status:** Complete

## Overview

Version 0.3.0 transforms Vintage Beef into a living, breathing world with procedural terrain generation, multiple biomes, resource gathering, inventory management, dynamic day/night cycles, and weather systems. This release focuses on creating an immersive exploration experience.

## What's New

### üåç Procedural World Generation

**ProceduralWorldGenerator.cs**
- Multi-octave Perlin noise terrain generation
- Chunk-based world system (20x20 chunks)
- Configurable world size (default 200x200 units)
- Dynamic mesh generation with vertex colors
- Automatic collider creation for physics
- Seed-based generation for reproducibility
- Performance optimized with ~400 triangles per chunk

**Technical Features:**
- Height multiplier and noise scale customization
- Three octaves of noise for terrain detail
- Smooth terrain transitions
- MeshFilter and MeshRenderer components
- MeshCollider for player interaction

### üé® Multiple Biomes

Four distinct biomes with unique characteristics:

**Forest Biome:**
- Deep green coloring
- 2x tree density
- 1.5x plant density
- Lush vegetation

**Plains Biome:**
- Light green/yellow grass
- Balanced resource distribution
- 2x plant density
- Open exploration

**Desert Biome:**
- Sandy tan/yellow terrain
- 2x rock density
- 0.2x tree density (sparse)
- Cactus fruit instead of herbs

**Mountains Biome:**
- Gray rocky terrain
- 3x rock density (iron ore)
- 0.5x tree density
- Elevated peaks

**Biome System:**
- Temperature and moisture-based selection
- Smooth biome transitions using Perlin noise
- Biome-specific resource drops
- Stylized Palia-inspired colors

### üå≤ Resource Gathering System

**ResourceNode.cs** - Interactive resources
- Three resource types: Trees, Rocks, Plants
- Hit point system (3 hits to deplete)
- 3-meter gather radius
- Press 'E' to gather
- Visual feedback (scales down)
- 60-second respawn timer
- Biome-specific drops:
  - **Trees**: Wood (2-5 pieces)
  - **Rocks**: Stone/Iron Ore (1-3 pieces)
  - **Plants**: Herbs/Cactus Fruit (1-2 pieces)

**Resource Features:**
- Automatic respawn system
- Hide/show on depletion/respawn
- Collider management
- Debug gizmo for gather radius
- Procedural placement per chunk

### üéí Inventory System

**PlayerInventory.cs** - Complete item management
- 30 inventory slots
- Stack system (99 items per stack)
- Automatic stacking with existing items
- Item categorization (material, consumable)
- Full inventory protection
- Add/Remove methods
- Query system:
  - `GetItemCount(itemName)`
  - `HasItem(itemName, amount)`
  - `GetAllItems()`
  - `GetItemsByType(type)`
- Empty/used slot tracking
- Print inventory for debugging

**InventoryUI.cs** - Visual interface
- Toggle with 'I' key
- Grid-based slot display
- Dynamic slot creation
- Item name and quantity display
- Disables player controls while open
- Auto-refresh on changes
- Palia-style clean interface
- Slots per row configuration (default 6)

### üåÖ Day/Night Cycle

**DayNightCycle.cs** - Dynamic time system
- 24-hour cycle (configurable duration, default 20 min)
- Sun and moon directional lights
- Automatic light rotation
- Dynamic color gradients:
  - Sun color (night blue ‚Üí dawn orange ‚Üí day yellow ‚Üí dusk orange ‚Üí night blue)
  - Ambient lighting progression
  - Fog color matching time of day
- Intensity curves for realistic transitions
- Smooth sunrise at 6 AM
- Smooth sunset at 6 PM
- Configurable start time
- Pause time functionality

**Features:**
- `GetTimeOfDay()` - Current hour (0-24)
- `SetTimeOfDay(hour)` - Set specific time
- `IsDaytime()` / `IsNighttime()` - Time queries
- `GetFormattedTime()` - 12-hour format display
- `SetPauseTime(bool)` - Pause/resume
- Sun active: 5 AM - 7 PM
- Moon active: 7 PM - 5 AM
- Soft shadows from both lights

### ‚õàÔ∏è Weather System

**WeatherSystem.cs** - Dynamic atmospheric conditions
- Three weather types: Clear, Rain, Foggy
- Automatic weather changes (60-second intervals)
- Smooth transitions (10 seconds)
- Probability-based selection:
  - 30% chance rain
  - 20% chance fog
  - 50% chance clear

**Rain Effects:**
- Particle system (1000 particles)
- Vertical rain droplets
- Semi-transparent particles
- Rain audio support (looping)
- Increased fog during rain
- Smooth fade in/out

**Fog Effects:**
- Heavy fog density (5x normal)
- Adjusted fog color
- Smooth density transitions
- Works with day/night cycle

**Weather API:**
- `GetCurrentWeather()` - Current type
- `SetWeather(type)` - Request change
- `ForceWeather(type)` - Instant change
- `IsRaining()` / `IsFoggy()` - Queries
- Configurable check intervals
- Transition progress tracking

## Gameplay Features

### Exploration
- Explore procedurally generated worlds
- Discover 4 distinct biomes
- Find resources appropriate to each biome
- Experience dynamic time and weather

### Resource Gathering
1. Locate resource nodes (trees, rocks, plants)
2. Approach within 3 meters
3. Press 'E' to gather (3 times)
4. Resource depletes and respawns after 60 seconds
5. Items automatically added to inventory

### Inventory Management
1. Press 'I' to open inventory
2. View all gathered items organized by slots
3. See quantities and item names
4. Items automatically stack up to 99
5. Track materials and consumables
6. Press 'I' to close and resume gameplay

### Time and Weather
- Watch sun rise and set throughout the day
- Experience different lighting at different times
- Weather changes automatically
- Rain and fog create atmospheric moments
- All effects work together seamlessly

## Setup Instructions

### Basic Setup
1. Open Unity Editor
2. Add ProceduralWorldGenerator to scene instead of SimpleWorldGenerator
3. Configure world size and seed
4. Add DayNightCycle component to Environment GameObject
5. Add WeatherSystem component to same GameObject
6. Play!

### Advanced Configuration
- Adjust biome scales for different terrain
- Modify resource counts per chunk
- Change day length for faster/slower cycles
- Adjust weather probabilities
- Add custom audio clips for rain/wind
- Customize color gradients for visual style
